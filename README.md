This project serves as a space for learning and developing my Bootstrap skills.

### Installation
In your terminal:
1. Clone the repo-
`git@github.com:Kristiannmiller/Ideabox-Bootstrap.git`
2. Install NPM packages- `npm install`
3. Start application- `npm start`
